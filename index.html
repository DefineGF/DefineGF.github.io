社会的发展并不会突破生产力划定的上限

农业社会中，上层需要征收绝大多数的农业剩余产品，用以供养贵族、官僚、士兵、僧侣、诗人等等，各个文明的农民几乎都过着仅以温饱的生活。

工业革命之前，技术对农业产出的影响微乎其微，劳动力与耕种土地遵循效益递减的原则：单位耕种的土地的产出不会随着投入劳动力的增加而增加。

同时，如马尔萨斯所言，人类和绝大多数生物一样，具有无限繁衍后代的趋向。一旦人口超过土地承载能力，相应的粮食产量并不会增加，社会积攒的矛盾便会如火药桶一样等待着随时迸发出的火星。

此时，统治者的几乎没有选择的余地：大规模的移民成本极高（湖广填四川）；雇佣过剩流民对下层镇压，不过是饮鸩止渴. . . 直到内战、瘟疫、外族入侵等等消耗掉过多的人口，存活下来的农民在过剩的土地上重建社会，于是盛世来了，又一个周期开始了。如五胡乱华、五代十国割据等，中国人朴素的盛极而衰的道理就在其中。同样，欧洲通过十字军东征无意间排除了剩余军事贵族这个定时炸弹；黑死病也无差别的消灭各个阶级的剩余人口，包括后来的航海时代，船员在航程中的报销率依旧高达50%。

这也是为什么“重农抑商”的儒家在东亚的农业社会具有举足轻重的地位。虽然商人在一定程度上优化经济配置，但这样的社会如果遇到偶然的天灾、外族意外的入侵或者经济的微小波动，那么将会有相当一部分人失去生活保障，同时也会有一小部分活不下去的会拿自己的性命和统治阶级开玩笑。短暂的高效不是最重要的，稳定才是。

科学技术二百多年来指数爆炸式增长

18世纪60年代，英国率先兴起工业革命。除了英国具有丰富的自然资源（浅层煤矿、铁矿，木材缺乏促进焦煤使用）、设立发明专利权、掠夺殖民地原料、教会的腐败促进科学技术的发展等等，还有更加独特的因素：

首先是农业革命：由于温带大陆地区的西风漂流带，欧亚大陆越向西，水源和肥沃的农场就越多，同时自美洲引进的农作物提高了单位耕地的产量，因此能够供养大批非农业人口；

其次是欧洲（尤其是西欧）在历次抵抗蛮族入侵时，形成了向外扩张的上层建筑，同时欧洲主要农业地区夹在波罗的海与地中海两个内陆的海域之间，为以后大洋航行积累了丰富的技术、人力基础；

最后是美洲近乎空白的资源宝地，在提供原材料同时，吸引了大批来自英国、西班牙等剩余人口，也给工业品提供了倾销市场。

当然，船大并不好调头：最大的农业国清朝由于过多的剩余人口，贸然采用节省人力的先进技术，比如蒸汽机代替纤夫，铁路代替骡运，很有可能造成大量不安定的流民。当然，劳动力充足的社会也不会主动地寻求技术突破，直到有一个改变大船运动状态的巨大的力。

从物质不灭的角度来讲，人类只是在做两件事，一是改变物质的位置，二是让别人也这样做（罗素，原话记不清了）。通俗理解就是，无论人消耗掉什么，物质都只是改变为另一种形态，哪怕是人死了，也是转换成各式各样的原子组成的物质归还世界。从能量守恒角度来讲，能量只是在人体里走了一圈，又转换成其他能量的形式。于是，就用熵的形式描述人类活动吧！负熵可简单理解为可利用能源数量的物理量（比如太阳能、化石能源等等）。

工业革命直观来讲就是提高了人类收集和利用负熵的效率！负熵源从农业社会中植物从太阳能转化而来的低熵化学物质，到工业社会中石油等化石能源。从微观角度，人类只是利用这些负熵将各种物质移动相对位置，比如利用能量从沙子中提取硅单质制作芯片，和宏观上将一块石头从一个地方搬到另一个地方本质没什么区别。而“创造”出来的有衣服、汽车等消费品，即经济学家口中的财富。

工业社会创造的财富可以供养更多的非农业者，用于促进文化、科技等等各个层次繁荣。假如康熙时期的普通人穿越到100年后的乾隆时代，不过两天这个人就能适应新社会的七七八八；假如民国的普通人穿越到100年后的现在，这般”离奇“世界可能远超他想象中的天国，这便是二百多年的爆炸式发展的工业的底蕴。

经济基础决定上层建筑。当人类的认知还未触及到分子层面时，阴阳五行解释世界勉强能蒙混过关；但现代医学发展的当下程度，再用肾属水之类的药方给人看病，甚至连自己的祖宗都瞧不上，起码作古的他们也知道吸纳外来文化（比如西域传来的琵琶，葡萄牙传来的佛郎机等）。一些一边夸耀着沐浴现代社会的荣光，骨子里还是封建地主的小家子气，一遇到社会变革，便匆忙地从封建社会中找统序。最顽固的保守派，一定是最激进变革的导演：英国能在对外扩张中时刻修正社会制度；而法国作为内陆国，只能通过内部解决矛盾，最终爆发了欧洲烈度最大的革命，贵族阶层在革命中被消灭殆尽。随着生产力的提高，所有生产关系正在被破坏， 它已经被破坏了！

科学技术是否一直会指数增长？

《技术奇点》似乎很乐观。但是我们应注意到，随着各领域的细分，很难再有像蒸汽机那样被广泛应用到各行各业、并提供正反馈的发明了；摩尔定律似乎撞到了物理极限，量子隧穿大概是传统芯片工艺未来的坟墓；可能再也不会出现像牛顿、爱因斯坦等一流科学家凭一己之力带动整个人类科学的发展，反而是越来越高的研究投入如无底洞一般让人类觉得走错了方向，当然还有横亘在科学家之间的政治正确；人们寄予厚望的人工智能，核心的数学分析早在百年前就开始了，当前研究员只是幸运的有了前人难以企及的算力和数据规模，同时也有了各式各样的商业需求。

生产力停滞能给工业国带来什么？一旦扩张的步伐被打乱，不像上述中工业革命的英国那样有广阔的殖民地用于原料掠夺和不平等关税保证的工业品倾销，也不像后冷战的美国那样开着航母满世界的建军事基地（美元霸权）；同时用于扩张的机构成为社会包袱（类似于农业社会的商人），社会趋于保守等等。可以简单理解市场状况：商品价格 = 利润 + 工资 + 其余成本，即商品价格 > 工资。当经济普遍停滞时，外部市场的萎缩使得内部工业品堆积，必然会导致生产过剩。然后无非是过剩生产再投资，或者粗暴的销毁商品解雇工人，循环开始。

俄乌、佩洛西、疫情等接连反转的2022年过去了，我会很怀念它。

brk-sbrk

Posted on 2023-05-18 | In c
进程内存布局

brk 和 sbrk

函数原型
brk

程序中断点（program break）初始化值是未初始化的数据段的结束位置；

当brk参数中 addr 值合理并未超过最大允许范围时，将 program break 值设为 addr；

返回值：

  • 0：成功！
  • -1：失败，并设置 error 值为 ENOMEM；
sbrk

通过函数参数 increment 设置程序中断点的移动范围，当 increment值：

  • 正：中断点向后移动，返回值为移动前位置；
  • 0：返回当前中断点位置；
  • 负：向前移动中断点，相当于释放内存，返回值无意义！
使用

一般使用 sbrk 分配内存，使用brk释放内存。

运行过程
  • 由 VM 负责创建对象，并将对象压入维护对象指针的栈中；用不到的对象（需要被回收）出栈；
  • VM 维护列表头，用以记录经由 VM 创建的所有对象（包括已经栈中存放的）；
  • 触发 gc 时候，通过遍历列表，标记存放在栈中的对象（不回收），然后调用 sweep 执行回收内存的操作！
